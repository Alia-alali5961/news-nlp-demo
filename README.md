## Features

- **Language Detection**: Automatically detects Arabic or English text
- **POS Tagging**: Part-of-speech tagging for both languages
- **N-gram Analysis**: Extracts and displays top bigrams
- **Perplexity Calculation**: Language model perplexity scoring
- **Named Entity Recognition**: Identifies entities in text

## Demo

Try the live demo: [News NLP Web Demo](https://tinyurl.com/news-nlp-demo)

## Technologies

- **Streamlit**: Web application framework
- **spaCy**: English NLP processing
- **NLTK**: Natural language toolkit
- **Python 3.11+**

## Local Installation

```bash
pip install -r requirements.txt
python -m nltk.downloader punkt punkt_tab averaged_perceptron_tagger averaged_perceptron_tagger_eng
streamlit run streamlit_app.py
```
